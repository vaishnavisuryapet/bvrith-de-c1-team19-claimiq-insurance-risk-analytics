# Weekly Log Templates

Each week has a ready log file.

Students update the current week's file only.

Example:

```text
Week 1 -> weekly_logs/week01_log.md
Week 2 -> weekly_logs/week02_log.md
...
Week 12 -> weekly_logs/week12_log.md
```

Each weekly log includes:

- Sprint goal
- Work completed
- Key decisions
- Blockers
- Evidence added to GitHub
- AI transparency note
- Next week preparation
